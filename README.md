# Nikah Project Vue.js

<a href="https://github.com/SorwarHussain/nikah-laravel">Backend Repositories</a>

## How to install

```
mkdir nikah-vue
```

```
cd nikah-vue
```

```
git clone git@github.com:SorwarHussain/nikah-vue.git client
```

```
cd client
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
